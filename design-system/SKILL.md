# R³AP Design System — Modern Studio

A guided classroom-culture operating system for educators. Studio Cypher aesthetic (tracks, mixing, equalizers, vinyl) executed as clean, utilitarian software. Teal depth + Orange signal. Inter Tight + JetBrains Mono.

## Load order
1. Read `README.md` first — full brand context, content fundamentals, visual foundations, iconography, and asset manifest.
2. Link `colors_and_type.css` for all design tokens (teal depth, signal colors, type, spacing, radii, elevation, glass) and primitive component classes (`btn-primary`, `btn-secondary`, `btn-ghost`, `card`, `card-action`, `card-growth`, `card-warn`, `glass-panel`, `glass-deep`).
3. Browse `preview/` for token + component cards (mirrored in the Design System tab).
4. See `ui_kits/teacher-app/` for the high-fidelity recreation of the 10-screen Teacher app.
5. Use `assets/` for brand imagery — coach portraits, studio props, mixtape concept art, classroom photography.

## When designing in this system
- Teal is **architecture** — use all 5 levels for hierarchy, never flat.
- Orange Pop is **reserved for action** — CTAs, active states, SAY THIS, Live mic pulse. Never decorative.
- Canvas is **#F8FAFC cool-slate**, cards are **#FFFFFF pure white**. Zero warm/cream backgrounds.
- Live screen is **one screen, 6 zones, no scroll**. SAY THIS ≤ 12 words.
- Permanent Marker font is **PLURAL State + milestone unlocks ONLY**.
- "Grade-Band Response Guidance" — never "Developmental Truths."
- No animals. No decorative emoji. Studio software, not toy.
