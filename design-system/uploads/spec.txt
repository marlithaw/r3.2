0 · DESIGN SYSTEM — MODERN STUDIO

━━━━━━━━━━━━━━━━



0.1 · Design Philosophy

The R³AP app merges high-end coaching intelligence with the "Studio Cypher" aesthetic: music production metaphors (tracks, mixing, equalizers) executed through clean, utilitarian software design. High contrast, sharp edges, and clear state changes make the app feel like premium audio production software — not a generic wellness tool.


AESTHETIC MANDATE:  Modern Studio. Teal depth system. Orange signal energy. Pure surfaces, sharp borders, stark typography. Every screen should feel like the control room of a world-class studio — the place where precision meets creativity.

1 · GOVERNING PRODUCT RULES

━━━━━━━━━━━━━━━━

Use the language already built. Do not add new terms unless a true gap exists. The app should feel like R³AP — not like a generic SEL, wellness, coaching, or classroom management platform.


CORE PRODUCT STATEMENT:  R³AP is a guided classroom-culture operating system that helps educators clear the fog, respond with precision under pressure, and build learning environments where regulation, belonging, and academic growth can be sustained.


SHORT POSITIONING:  R³AP helps educators clear the fog in the moment, choose the next grounded move, and build classroom cultures where students can regulate, belong, and grow.





1.1 · Cultural Operating System Pillars
PILLAR
DEFINITION & EVIDENCE TARGETS
Regulation
Adults stabilize themselves and the environment so students can enter a learning-ready state. Evidence: reduced time to readiness, fewer repeated redirects, increased use of precise teacher language.
Belonging
Students experience clear expectations, fair systems, dignity, structured participation, and repair. Evidence: increased participation, more dignity-preserving corrections, more repair conversations.
Growth
Students move from readiness and participation into stronger academic behavior and measurable learning progress. Growth must include student academic outcomes — not just engagement. Evidence: increased work completion, improved exit ticket performance, more instructional time protected.





1.2 · R³ Internal Change Process
STAGE
DEFINITION
Reflection
See the moment clearly enough to understand what it is showing.
Resistance
Interrupt the automatic pattern before it takes over.
Revelation
Integrate the new understanding and practice the new response until it becomes part of the adult&apos;s operating system.

4 · CORE SYSTEM LOOP — DAILY READINESS ENGINE

━━━━━━━━━━━━━━━━

Every morning: a maximum 5-minute loop producing three measurable system outputs.


THREE OUTPUTS:  Teacher Regulation State (TRS)  ·  Student Readiness State (SRS)  ·  Instructional Readiness Time (IRT)





4.1 · Morning Pulse

  FEATURE · MORNING PULSE  



3-tap teacher + class check-in generating an instant readiness score. Target: 30–60 seconds.
COMPONENT
DESIGN SPEC
Teacher State Selector
Three full-width tap cards on Teal Dark background: CALM (Teal Mid fill) · TIGHT (Amber fill) · OVERWHELMED (Orange fill). JetBrains Mono labels. No icons needed — state is visually self-evident through color.
Readiness Score Dial
Circular gauge. Orange Pop fill arc on Teal Deep track. Score number in Inter Tight ExtraBold 48px, white. Percentage delta badge in Teal Surface chip.
Auto-Routing
If OVERWHELMED: coaching card triggers before routing. If TIGHT/CALM: routes to recommended next screen.
Output
TRS logged. Balance Period fog check prompted if OVERWHELMED state occurs 3+ consecutive days.





4.2 · Time to Learn Tracker

  FEATURE · TIME TO LEARN TRACKER  



Teacher taps START when instruction actually begins. App auto-calculates delay from class start time and trends it in Master the Flow.
COMPONENT
DESIGN SPEC
Timer FAB
Floating action button. Orange Pop fill. Label: "Instruction Started." Placed bottom-right on Live and Clear the Fog screens. Pressing it logs IRT timestamp.
Fast Start Score
Green badge: < 3 min to readiness · Yellow badge: 3–6 min · Orange badge: 6+ min. Displayed in Liner Notes summary and Master the Flow.
Pattern Statement
Auto-generated in "Let&apos;s R.A.P. About It" after 5+ data points: "On days you grounded first, your class reached readiness 3 minutes faster."





4.3 · Pattern Finder

  FEATURE · PATTERN FINDER  



Weekly summary auto-surfaces in Master the Flow and coaching cards. Analyzes Liner Notes to surface behavioral and classroom readiness trends.
PATTERN TYPE
EXAMPLE OUTPUT
Day-of-week patterns
"Mondays = highest dysregulation. Fridays = strongest participation."
Teacher state correlation
"After teacher grounding logged as CALM: classes reached readiness 38% faster."
Student pattern frequency
"Refusing routine appeared 7 times this week across 3 class periods."
Track effectiveness
"Entry Routine track: time-to-readiness dropped from 9 min → 5 min in 2 weeks."





4.4 · Script Engine

  FEATURE · SCRIPT ENGINE  



Powers "Say This / Do Not Say This" logic across Live, Practice Station, and Playbook. Exact language + tone guidance + follow-through.
COMPONENT
SPEC
SAY THIS
Orange Pop left border card on white. Inter Tight SemiBold 18px. ONE line maximum. Example: "Your seat is posted. Find it."
DO NOT SAY
Red left border card. JetBrains Mono strikethrough text. Example: "Why do you always do this?"
Tone Guidance
JetBrains Mono pill tag: FLAT · CALM · FIRM · WARM. Teal Surface fill, Teal Mid text.
Follow-Through
One clear action. Teal Bright italic text. Arrow icon. Example: "Return attention to the class. Follow up privately."

08 Remix

─────────────────────────────

  TEACHER LANGUAGE: "REMIX. REPLACE THE OLD PATTERN WITH A STRONGER TRACK."  




Four Remix Modes
MODE
WHAT IT DOES + UI SPEC
A · Rewrite the Move
Turn a rough, reactive, or overexplained response into precise teacher language. Input form: 4 voice/text fields (white cards, Teal Mid top border). Output card: Stronger line (Orange left border), Shorter version, Do Not Say (Red border), Follow-up option.

Turn a repeated classroom breakdown into a routine students can follow. Inputs: 6 prompts. Output: Student steps + Teacher script + Visual cue + Practice moment + Liner Note prompt. Displayed as a numbered steps card (JetBrains Mono numerals in Amber).

Convert a Current Track or reflection into a song, chant, hook, or call-and-response. Teacher selects: Track · Audience · Purpose · Tone · Key phrases. Output: Lyric draft + Hook + Chant + Suno-ready prompt. OPTIONAL — not required. Legal guardrail card displayed on entry.

Revisit any failed Live moment, scenario, Playbook routine, Liner Note reflection, or work-completion problem. Each revisit logged for Pattern Finder. Orange "↺" icon. Normalizes repetition as professional practice, not failure.


LEGAL GUARDRAIL (displayed on Put It on a Track entry):  Before public or commercial use of AI-generated songs, verify current tool terms and licensing. This track is for personal and classroom use.

9.3 · Aware vs. Attentive — Governance Rule


DO NOT DEVIATE FROM THESE DEFINITIONS:  Aware = current-moment orientation ("What is actually happening right now?") — NOT the wellness load. 11 · NON-NEGOTIABLE CONSTRAINTS

━━━━━━━━━━━━━━━━



For the Visual Designer
CONSTRAINT
RULE
App Canvas Background
Always #F8FAFC (cool-slate). NEVER cream, warm white, or off-white. Cards are pure #FFFFFF.
No Warm Backgrounds
Zero bronze, tan, sepia, or cream (#F4EDE3) tones anywhere in the app. These belong to the marketing site only.
Teal as Depth
Use all five teal levels. Never flatten teal to a single color. The depth system creates visual hierarchy.
Orange = Action
#FF6B00 is reserved for primary actions, active states, and signal elements ONLY. Not decorative.
Amber = Warmth/Secondary
#FF9800 for hover states, gradient ends, warm accents, and track numerals. Not primary.
Red = Warning Only
#B8423D for "Do Not Say" and error states ONLY. Never used decoratively.
Permanent Marker Font
ONLY for PLURAL State celebration text and milestone unlocks. Nowhere else.
No Animals
No animals anywhere — not as icons, mascots, metaphors, or decorative elements.
No Generic AI Imagery
All imagery: polished, 3D-adjacent, hip-hop-culture-aligned. Coach avatars are high-fidelity 3D portraits.
Live Screen
Always ONE line, ONE Do Not Say, ONE next move, ONE return cue, ONE documentation prompt. Nothing more.
Balance Period Placement
Balance belongs ONLY in: Clear the Fog, The Cypher, Master the Flow.
Grade-Band Label
Always "Grade-Band Response Guidance." Never "Grade-Band Developmental Truths."
Current Track Limit
One active track per teacher at all times. No multi-track loading.
Admin Privacy
Admin cannot see individual private teacher reflections or wellness entries without explicit pilot agreement.
